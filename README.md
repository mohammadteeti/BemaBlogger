# BemaBlogger
a personal blogger application

__Bema__ is a personal web  blogger  that categorizes user posts under different fields 

### Bema allows to : 
  * Register new User 
  * Login/Logout
  * Reset/Change password
  * Post in any category 
  * comment on other users posts 

# Requirements :
  * Django v 4.1
